## Instalar dependencias
Antes de ejecutar la aplicacion se deben intalar las dependencias. Para esto ejecute:

```bash
npm install
```

o 

```bash
sudo npm install
```

## Ejecutar la aplicación
Para ejecutar la aplicación, ejecute:

```bash
sudo ng serve
```
